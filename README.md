# TaskFlow - Task & Project Tracker

TaskFlow is a web application that allows users to create projects, add tasks, and track progress.

## Tech Stack

- **Frontend:** React + Bootstrap
- **Backend:** Node.js + Express
- **Database:** MySQL
- **Source Control:** Git + GitHub
- **Deployment:** Azure (via GitHub Actions)

## Folder Structure

- `backend/` - Server-side API built with Node.js and Express
- `frontend/` - Client-side React application
